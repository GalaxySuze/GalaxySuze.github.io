Postman使用json格式测试API

  1. 2019-11-12
  2. 80字
  3. 1 min read

导入cURL

  1. 打开浏览器开发工具 Network
  2. 右键选择需要请求的URL，Copy -> Copy as cURL
  3. 打开Postman，选择左上方 Import -> Paste Raw Text，点击 Import 即可

手动填入

  1. 选择 POST 请求方式
  2. 设置 headers 增加 「Content-Type」「application/json」
  3. 设置 body 为「raw」
  4. 填入 json 数据发送测试
